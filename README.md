# Lux by Nodus

Peer-validated protocol turning lived experience from neurodivergent families into a community-owned living gap map.

## Live demo
https://lux-nodus.vercel.app

## What it does
Lux is a community signal board where families and practitioners submit voice-enabled observations about what works in neurodivergent care. AI clusters the signals in real time and surfaces a living gap map — showing what the community knows and where answers are still missing. Every contributor earns a Solana token as permanent proof of their contribution.

## Stack
- React + TypeScript
- Supabase (database + edge functions)
- Claude API (signal clustering and quality review)
- Web Speech API (voice input)
- Solana (wallet connection + token mechanics)
- Noah AI (development platform)
